Keshab Jewellery House

 Project Overview

Keshab Jewellery House is a modern MVP (Minimum Viable Product) jewellery website designed to showcase premium jewellery collections with an elegant and interactive user experience.

The website focuses on clean UI, smooth animations, and 3D visual elements to attract users and enhance product presentation.

---

 Live Demo

🔗 Live Website: https://shrabaniadhikari03.github.io/Jewellery-website

---

 Features

*  Elegant Jewellery Showcase
*  Fully Responsive Design
*  Fast Loading Performance
*  Smooth Animations using Framer Motion
*  Interactive 3D Elements using Three.js
*  Basic Product Display (MVP Level)

---

Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap
* Three.js
* Framer Motion

 Installation & Setup

1. Clone the repository:
   git clone https://github.com/shrabaniadhikari03/Jewellery-website.git

2. Open the project folder:
   cd keshab-jewellery-house

3. Run the project:
   Open index.html in your browser
